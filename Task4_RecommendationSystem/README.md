# Task 4 â€“ Recommendation System

This project demonstrates a simple Recommendation System built in Java.  
It recommends movies to a user based on shared preferences with other users.

## Features
- Uses Java Collections to simulate a dataset
- Calculates user similarity based on common movie interests
- Suggests new movies based on the most similar user

## Example Output
User: Ajith  
Liked Movies: [Ala Vaikunthapurramuloo, Pushpa , Arya, S/O Satyamurthy]  
Recommended Movies: [Captain America, Thor]

## How to Run
1. Open terminal in project folder.
2. Compile:
   javac Task4_RecommendationSystem/RecommendationSystem.java
3. Run:
   java Task4_RecommendationSystem.RecommendationSystem
